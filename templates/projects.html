My Projects

Echidna.py

A Discord bot written in Python. The application serves as a general purpose utility for many servers, including commands designed for math, image processing, anime and text-based mini-games.

What I learned:

  • Asynchronous programming (coroutines, async/await)
  • Making API requests (GET/POST)
  • Wrapper API libraries around mainstream web applications
  • Writing modular code and effectively organizing packages
  • Reading and writing JSON data and its use in RESTful applications

UniMeetups

A web application designed for CalgaryHacks 2022. This website aimed to connect university students across Canada, with a more anonymized approach to the user experience. Users can register their account with their intended school, and they will be matched with other users in the school hub that have similar interests. If a user finds a potential match interesting, they can send a meetup request to the end user, who may accept or decline their adavances. There is also a section dedicated to public meetup events, where any user enrolled in a school's meetup board can create their own events.

What I learned:

  • Working in a collaborative software setting and using VCS effectively.
  • Website routing
  • Front and back end communication
  • Using server side databases and application caching
  • User authentication protocols

Tunes.Java

A desktop music player, designed using the Swing GUI framework written in Java. It was created as my final project, within my object oriented programming course. Users can open their own mp3 files and create/save/edit their own playlists.

What I learned:

  • Making responsive layouts on a desktop application
  • Applying OOP and design practices in software development
  • Serializing the state of an application on a user's system locally
